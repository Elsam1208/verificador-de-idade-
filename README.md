📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em desenvolvimento web, utilizando HTML, CSS e JavaScript. A aplicação apresenta uma interface moderna, responsiva e funcional, focada na organização e exibição de informações de forma clara e intuitiva.

🚀 Funcionalidades
Interface amigável e de fácil navegação
Layout responsivo para diferentes dispositivos
Manipulação de dados com JavaScript
Estrutura organizada e código limpo
Componentes reutilizáveis
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript
📂 Estrutura do Projeto
index.html → Estrutura principal da aplicação
style.css → Estilização da interface
script.js → Lógica e interações
imagens 
🎯 Objetivo

O principal objetivo deste projeto é consolidar conhecimentos em desenvolvimento front-end, além de servir como base para projetos mais avançados.
